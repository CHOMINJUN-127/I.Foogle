# I.Foogle 🔍

🌐 **Live Demo:** https://chominjun-127.github.io/I.Foogle/

---

## ▶️ 실행 방법

브라우저에서 상단 링크를 클릭하여 바로 실행할 수 있습니다.



## 📌 프로젝트 소개

I.Foogle은 Google 검색 페이지의 UI를 HTML/CSS로 구현한 프론트엔드 프로젝트입니다.  
사용자가 검색어를 입력하면 실제 Google 검색 결과로 연결되며,  
인공지능, 파이썬, 과학 뉴스 등의 키워드를 버튼으로 제공해  
빠르게 검색할 수 있도록 설계되었습니다.

이 프로젝트를 통해 HTML/CSS 레이아웃 설계와  
JavaScript를 활용한 버튼 이벤트 처리 경험을 쌓는 것을 목표로 했습니다.



## 🎯 제작 목적

코딩을 시작한 지 얼마 되지 않아, HTML과 CSS, 간단한 JavaScript를 활용하여 실습 겸 프로젝트 경험을 쌓고자 제작했습니다.  
이 프로젝트는 자율동아리 활동에서 시작되었으며, 동아리에는 컴퓨터공학이나 코딩에 관심 있는 학생뿐만 아니라 경영 분야를 목표로 한 친구들도 함께 참여했습니다.  

I.Foogle이라는 이름은 'I'가 Information(정보), 'F'가 Finger(손가락)을 의미하며,  
경영이든 코딩이든 정보를 다루려면 결국 손가락으로 입력하고 활용해야 한다는 의미에서 '정보 손가락'이라는 이름이 나오게 되었습니다.  
동아리 활동에서 진행한 ESG 경영 관련 페이지와 연계되는 **메인 페이지 느낌**을 주기 위해 만들어졌습니다.  

현재는 기존 동아리 코드가 사라져, I.Foogle만 다시 재구현한 형태입니다.



## ✨ 주요 기능

- 입력한 검색어를 Google 검색 결과로 연결하는 기능
- 인공지능, 파이썬, 과학 뉴스 버튼을 눌러 빠르게 검색 가능
- Google 검색 페이지 스타일을 재현한 UI
- HTML과 CSS를 활용한 레이아웃 구성
- JavaScript를 활용한 버튼 클릭 이벤트 처리

## 🛠 사용 기술

- HTML5: 페이지 구조 및 폼 구성
- CSS3: 레이아웃, 색상, 폰트, 반응형 디자인
- JavaScript: 버튼 클릭 이벤트 및 검색어 자동 입력 처리
- Google Fonts (Roboto): 글꼴 스타일 적용


## ⚠️ 주의사항

- I.Foogle에서 입력한 검색어는 실제 Google 검색으로 연결됩니다.  
  따라서 I.Foogle 자체적으로 검색 결과를 제공하지 않습니다.
- 정적 웹 페이지이므로 서버 설치나 별도의 환경 설정 없이, 브라우저에서 바로 실행 가능합니다.
- 최신 브라우저(Chrome, Edge, Firefox 등)에서 최적화되어 있으며, 일부 구형 브라우저에서는 UI가 다르게 보일 수 있습니다.
